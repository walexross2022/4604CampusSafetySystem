Welcome

This system provides an interactive platform for exploring campus crime data collected from Virginia Tech public safety reports. Users can securely access incident records, analyze trends, and explore crime patterns across multiple years.

To begin, please log in or create an account. Authentication ensures secure access to the system and allows controlled interaction with the database.
