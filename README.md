# 4604CampusSafetySystem
The Campus Safety &amp; Incident Analytics System is designed to analyze, store, and manage campus crime data across multiple universities.

Design and implementation of a database-driven Campus Safety & Incident Analytics System that analyzes publicly available campus crime and safety reports. We could also scale this easily to include additional universities. Using monthly campus crime logs published by university police departments, the project will store, organize, and analyze anonymized incident data in a relational database.
The goal of the system is to help users explore patterns and trends in campus safety incidents through structured queries and a simple, user-friendly interface. Users will be able to filter incidents by crime type, date range, and location, view monthly and yearly trends, and generate summary reports. The project emphasizes database design, normalization, SQL querying, and ethical data use rather than advanced prediction or machine learning.
