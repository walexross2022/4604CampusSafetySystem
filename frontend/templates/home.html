About This Application

This system provides an interactive platform for exploring campus crime data collected from Virginia Tech public safety reports. Users can view, sort, and analyze incidents across multiple years, gaining insights into crime trends, locations, and patterns over time.

Key features include dynamic data tables, filtering by year, and built-in analytics such as most common offenses and peak incident times. This application demonstrates how data systems can support safety awareness and decision-making through structured reporting and visualization.

Authors: Alex Ross, Jasmine Alabi, Aastha Paranthaman, Gayatri Kamtala
